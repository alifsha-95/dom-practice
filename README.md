# DOM Manipulation and Events Practice

A practice project built as part of The Odin Project Foundations curriculum.

## What I Learned

- How to select HTML elements using querySelector
- How to create new HTML elements using createElement
- How to add elements to the page using appendChild
- How to style elements using JavaScript
- How to add event listeners to elements
- The difference between onclick and addEventListener

## What This Project Does

- Creates a red paragraph element
- Creates a blue h3 element
- Creates a div with black border and pink background
- Creates 3 buttons with click events

## Technologies Used

- HTML
- JavaScript
- DOM Manipulation
